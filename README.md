# Noise Sculptor

Interactive web app to explore different "colors" of noise by sculpting audio with filters.

## What it does
- X-axis: LPF â†” HPF cutoff frequency (left = low-pass, right = high-pass)
- Y-axis: Filter slope (-12dB to -48dB/octave)
- Creates different noise colors (brown, pink, white, blue, violet)
- Real-time audio with visual feedback

## Quick start
```
npm install
npm run dev
```

Touch or click to shape your sound.
